# AI Decision Support for EU Trade & Energy Risk

An interactive analytics dashboard built with Python and Streamlit to monitor trade disruption and energy risk across European countries.

## Live Dashboard
https://eu-trade-risk-dashboard.streamlit.app

## GitHub Repository
https://github.com/rajbhayani-sketch/eu-trade-risk-dashboard

## Project Overview
This dashboard analyzes trade risk and energy dependency across EU countries and calculates a combined risk score to identify high-risk regions.

The system allows users to:
- Explore country-level risk metrics
- Compare trade and energy risk
- Visualize risk distribution across Europe
- Interactively filter risk categories

## Key Features
- Interactive Streamlit dashboard
- EU risk scoring model
- Country comparison charts
- EU risk heatmap
- Risk category filtering
